OFDR_PINN_Demodulation 项目技术总结
更新时间：2026-03-09

========================================
1) 项目当前技术路线
========================================
- 任务：对全同 UWBFG 阵列局部谱进行解调，回归目标光栅的波长漂移 Delta-lambda（nm）。
- 总体链路：阵列光谱仿真 -> 局部窗提取与畸变注入 -> 模型/传统方法解调 -> 指标评估（RMSE/MAE/R2）。
- 当前论文主线模型：CNN+SE（1D CNN + Squeeze-and-Excitation）。


========================================
2) 仿真波形与数据构建参数（当前主配置）
========================================
配置来源：
- config/phase4a.yaml
- config/phase4_array.yaml
- src/phase4a/array_simulator.py
- src/phase4a/local_window.py
- src/phase4a/generate_dataset_phase4a.py

2.1 阵列与谱线基础参数
- n_gratings = 5（全同阵列，目标为第 3 个，index=2）
- 中心波长 lambda0 = 1550.0 nm
- 单栅谱线模型：Gaussian
  R(lambda) = A * exp(-0.5 * ((lambda-center)/sigma)^2)
- 基础谱宽 linewidth_sigma = 0.08 nm
- 基础幅值 amplitude = 1.0
- 波长采样范围：1549.0 ~ 1551.0 nm
- 输入点数：512（即网络输入长度固定为 512）

2.2 标签与扰动设置
- 目标栅漂移范围：Delta-lambda_target in [-0.2, 0.8] nm
- 邻居栅漂移：Delta-lambda_neighbors = 0.0（默认固定）

2.3 局部畸变机制（已实现）
- 目标窗随机偏移：[-0.02, +0.02] nm
- 邻居数量随机：neighbor_mode_options = [2, 4]
- 邻居泄漏非对称随机：
  - 一阶左邻：0.15 ~ 0.55
  - 一阶右邻：0.15 ~ 0.55
  - 二阶左邻：0.05 ~ 0.25
  - 二阶右邻：0.05 ~ 0.25
- 每个光栅的幅值与谱宽随机缩放：
  - amplitude_scale: 0.90 ~ 1.10
  - linewidth_scale: 0.90 ~ 1.10
- 平滑低频随机基线：
  - baseline_offset: -0.02 ~ 0.02
  - 低频分量数：2 ~ 4
  - 分量幅值：0.003 ~ 0.020
  - 频率范围：0.3 ~ 2.0（归一化坐标）
- 加性高斯噪声：noise_std = 0.005
- 归一化：minmax_per_sample
- 约束：clip_to_nonnegative = true

2.4 数据规模与划分
- 总样本数：8000
- 划分：train/val/test = 0.70/0.15/0.15
  - train = 5600
  - val = 1200
  - test = 1200
- 随机种子：seed = 42（数据生成主种子）

4) 最新结果（截至当前文件）
========================================
4.1 Step2 多随机种子汇总（results/paper_results_step2/final/multiseed_summary.csv）
单位：nm（MAE/RMSE）

- Cross-correlation: MAE=0.137153, RMSE=0.260904, R2=0.203566
- Parametric fitting: MAE=0.098739, RMSE=0.211443, R2=0.476911
- MLP: MAE=0.006554±0.000388, RMSE=0.009452±0.000284, R2=0.998954±0.000063
- CNN: MAE=0.005296±0.000232, RMSE=0.008469±0.000191, R2=0.999160±0.000038
- CNN+SE: MAE=0.005077±0.000195, RMSE=0.008367±0.000145, R2=0.999181±0.000028

结论：
- 传统方法明显落后于深度学习方法。
- CNN+SE 相比 CNN 存在小幅但稳定优势（误差均值更低、seed 波动更小）。
